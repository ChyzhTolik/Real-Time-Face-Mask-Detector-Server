Є папка з проектом, куди треба підключити мій клас. Для цього треба:
1.1. Відкрити мою папку SQLServer Package
1.2. В папках Debug і Relase лежать SQLServer.lib i SQLServer.dll для архітектури x86. Вибрати, яку версію ви використовуватимете.
1.3. У папку з проектом перекиньте SQLServer.dll 

2.1. Відкрити мою папку SQLServer Package
2.2. Відкрити папку SQLAPI
2.3. Відкрити папку bin
2.4. У папку з проектом перекиньте sqlapi.dll 

3.1. Відкрити папку SQLServer Package
3.2. У папку з проектом перекиньте папку SQLServer (це папка з проектом .dll, побудованим на архітектурі x86)

4.1. Зайти в налаштування ВАШОГО проекту (зверху в Visual Studio третій пункт зліва Project -> ВашПроект Properties.
4.2. У вікні, яке появилось, зверху буде вибір конфігурації і платформи. Виберіть All Configurations i All Platforms.
4.3. У цьому ж вікні перейти по Configuration Properties -> C/C++ (цей пункт появиться, якщо в проекті є хоча б 1 файл.cpp) -> General.
4.4. Найти пункт "Additional Include Directories" і вписати туди 2 шляхи до заголовкових файлів:
     SQLServer/SQLAPI/include
     SQLServer
     
5.1. У цьому ж вікні перейти по "Configuration Properties" -> Linker -> Input.
5.2 Найти пункт "Additional Dependencies" і вписати туди шляхи до двох бібліотек:
     SQLServer\SQLAPI\vs2019\lib\sqlapi.lib
     SQLServer\SQLServer.lib

6.1. У цьому ж вікні перейти по "Configuration Properties" -> Linker -> General.
6.2. Найти пункт "Additional Library Directories" і вписати туди шлях до бібліотеки:
     SQLServer\SQLAPI\vs2019\lib

// Наступний пункт виконується для тестування того, чи все нормально вийшло.
7.1. Відкрити мою папку SQLServer Package
7.2. В ній є файл TestCode. Скопіюйте його код в ваш файл і запустіть програму. Якщо вона успішно запуститься, значить ви все зробили нормально.
7.3. Якщо програма запуститься, то вона може вивести на екран повідомлення про те, що до бази даних не можливо підключитись. Змініть відповідні
     значення в початкових змінних, щоб перевірити вашу базу.
